## 编译步骤

### 使用Qt Creator
1. 打开 `propositional_system.pro`
2. 清理项目 (Build -> Clean All)
3. 运行 qmake (Build -> Run qmake)
4. 构建项目 (Build -> Build Project)
5. 运行 (Ctrl+R)

### 使用命令行
```bash
cd d:/codespace/system/propositional_system
# 清理
mingw32-make clean
# 重新运行qmake
qmake propositional_system.pro
# 编译
mingw32-make
```

## 可能遇到的问题

### SQLite驱动问题
如果提示 "Driver not loaded"，确保安装了SQLite驱动：
- 检查 `Qt/6.x.x/mingw_64/plugins/sqldrivers/qsqlite.dll` 是否存在

### OpenSSL问题
Windows下网络请求可能需要OpenSSL库，从Qt安装目录复制：
- `libssl-1_1-x64.dll`
- `libcrypto-1_1-x64.dll`

## 头文件引用规范

### 在src目录下的文件引用：
```cpp
// 引用dao目录下的头文件
#include "dao/userdao.h"
#include "dao/questiondao.h"

// 引用config目录下的头文件
#include "config/config.h"
#include "config/ai_config.h"

// 引用src根目录的头文件
#include "database.h"
#include "models.h"
```

- `database.h` - 数据库管理
- `models.h` - 数据模型（包含User、Question、Paper等）
