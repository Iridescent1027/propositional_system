# 离散数学命题系统 - 项目设计文档

## 1. 项目概述

### 1.1 项目目标
开发一个基于Qt框架的离散数学命题系统，支持题目管理、试卷生成、用户管理等功能，采用SQLite数据库存储，并集成AI技术实现智能化命题。

### 1.2 技术栈
- **前端框架**: Qt 6 (C++17, QWidget + Qt Designer)
- **构建系统**: qmake
- **数据库**: SQLite3
- **AI模块**: LLM API集成
- **富文本编辑**: QTextEdit / QWebEngine
- **PDF导出**: QPrinter

### 1.3 系统架构

## 3. 数据库设计

### 3.1 数据库表结构

#### 3.1.1 users (用户表)
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT UNIQUE NOT NULL,
    email TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    nickname TEXT,
    avatar TEXT,
    role INTEGER DEFAULT 1,     -- 1:普通用户, 2:管理员
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    updated_at TEXT DEFAULT (datetime('now', 'localtime'))
);

CREATE INDEX idx_users_username ON users(username);
CREATE INDEX idx_users_email ON users(email);
```

#### 3.1.2 questions (题目表)
```sql
CREATE TABLE questions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    type INTEGER NOT NULL,      -- 1:命题逻辑, 2:集合论, 3:图论, 4:关系
    content TEXT NOT NULL,      -- 题目内容
    options TEXT,               -- 选项(JSON格式)
    answer TEXT NOT NULL,       -- 标准答案
    explanation TEXT,           -- 答案解析
    difficulty INTEGER,         -- 难度等级 1-5
    knowledge_points TEXT,      -- 知识点(JSON数组)
    created_by INTEGER NOT NULL, -- 创建者ID
    is_ai_generated INTEGER DEFAULT 0, -- 是否AI生成(0/1)
    status INTEGER DEFAULT 1,   -- 1:正常, 2:审核中, 3:已删除
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    updated_at TEXT DEFAULT (datetime('now', 'localtime')),
    FOREIGN KEY (created_by) REFERENCES users(id)
);

CREATE INDEX idx_questions_type ON questions(type);
CREATE INDEX idx_questions_difficulty ON questions(difficulty);
CREATE INDEX idx_questions_created_by ON questions(created_by);
```

#### 3.1.3 papers (试卷表)
```sql
CREATE TABLE papers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    description TEXT,
    total_score INTEGER DEFAULT 100,
    config TEXT NOT NULL,       -- 试卷配置(JSON格式)
    user_id INTEGER NOT NULL,   -- 创建者ID
    status INTEGER DEFAULT 1,   -- 1:草稿, 2:已发布, 3:已归档
    ai_assisted INTEGER DEFAULT 0, -- 是否AI辅助
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    updated_at TEXT DEFAULT (datetime('now', 'localtime')),
    FOREIGN KEY (user_id) REFERENCES users(id)
);

CREATE INDEX idx_papers_user_id ON papers(user_id);
CREATE INDEX idx_papers_status ON papers(status);
```

#### 3.1.4 paper_questions (试卷题目关联表)
```sql
CREATE TABLE paper_questions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    paper_id INTEGER NOT NULL,
    question_id INTEGER NOT NULL,
    score INTEGER NOT NULL,     -- 该题分数
    order_num INTEGER,          -- 题目顺序
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    FOREIGN KEY (paper_id) REFERENCES papers(id) ON DELETE CASCADE,
    FOREIGN KEY (question_id) REFERENCES questions(id)
);

CREATE INDEX idx_paper_questions_paper_id ON paper_questions(paper_id);
CREATE INDEX idx_paper_questions_question_id ON paper_questions(question_id);
```

### 3.2 数据库关系图
```
users (1) ←── (N) questions
users (1) ←── (N) papers
papers (N) ←── (M) questions (通过paper_questions)
```
